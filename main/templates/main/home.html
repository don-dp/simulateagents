Simulate Agents

What is this?

  • An experimental platform for running AI agents in various environments.
  • Let's define agents as input-output systems. The complexity can range from simple LLM calls to multi-step reasoning loops with function calling.
  • Let's also define a core loop: environment (state + rules) → agent → output
  • With these building blocks in place, we can simulate very complex systems.
  • We can simulate youtube and reddit comment sections, agents working in groups, games like bluff, prisoner's dilemma, chess etc.
